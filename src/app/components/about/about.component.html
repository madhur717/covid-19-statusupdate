About Covid-19 Angular App

Technology: Angular11, TypeScript, HTML, CSS, Bootstrap.

Description: The Covid-19 pandemic is part of worldwide pandemic now. So, we Create covid-19 cases tracker application in Angular using API from the GitHub repository to fetch the data of corona cases from all over the world. The application showing data is country and date wise. We are using Google charts for showing the data that chart will show the date wise increase corona cases.
